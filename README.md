一款基于 **阿里云 DeepSeek R1** 的轻量化 AI Web 实用工具，实时分析商品品类，提供选购要点、避坑提醒和性价比技巧，帮助用户理性消费、规避网购陷阱。

## 功能介绍

### 🛒 商品品类检索
输入任意商品品类（手机、笔记本电脑、护肤品等），AI 实时分析并生成选购指南。

### 🤖 AI 智能分析
调用 DeepSeek R1 智能体，实时生成：
- **选购要点**：核心挑选标准和参数指标
- **避坑提醒**：常见营销套路和陷阱识别
- **性价比技巧**：价格对比和购买建议

**实时进度动画**：
- 使用 requestAnimationFrame 实现平滑进度条
- 缓动效果（ease-out），自然减速
- API 完成前停在 92%，完成后快速填充到 100%

### 🔮 AI 智能推荐
根据搜索历史自动推荐相关商品品类：
- **映射表优先**：预设 15+ 常见商品关联（手机→手机壳、充电器等）
- **AI 实时分析**：映射表无匹配时，调用 DeepSeek R1 实时推荐
- **本地缓存**：推荐结果缓存 1 小时，避免重复调用
- **加载动画**：AI 思考时显示"AI 正在思考推荐..."动画

### 🎤 语音输入
支持语音输入商品品类，自动转换为文字并填入搜索框。

**交互方式：**
- **桌面端**：点击麦克风按钮开始/停止录音
- **移动端**：长按麦克风按钮（约 0.4 秒），松开即可停止

**浏览器兼容性：**

| 浏览器 | 支持状态 | 说明 |
|--------|----------|------|
| Chrome（Android） | ✅ 完全支持 | 推荐浏览器 |
| Safari（iOS 14.5+） | ✅ 支持 | 需允许麦克风权限 |
| 微信内置浏览器 | ⚠️ 部分支持 | 建议切换系统浏览器 |
| UC浏览器 | ⚠️ 部分支持 | 建议切换到极速模式 |
| Firefox（移动端） | ❌ 不支持 | 自动隐藏麦克风按钮 |

**注意事项：**
- 语音识别需要 HTTPS 环境
- 不同浏览器对麦克风权限管理方式不同
- 支持自动去除标点符号

### 📷 一键截图
点击截图按钮，将选购指南保存为图片，方便分享和收藏。
- **桌面端**：直接下载图片文件
- **移动端**：弹出预览弹窗，长按保存到相册

### ⭐ 收藏功能
点击收藏按钮，将心仪的分析结果保存，支持查看和管理收藏。
- 最多保存 10 条收藏
- 右键删除单条或清除全部
- 收藏时保存原始查询时间

### 🎨 主题切换
点击右上角 ☀️/🌙 按钮，一键切换深色/浅色模式。
- 首次访问跟随系统偏好
- 自动保存用户偏好到 localStorage
- 0.8 秒平滑过渡动画

### 📜 搜索历史
自动记录最近搜索（最多 5 条），点击快速重新查询，右键删除单条或清除全部。

## 常见问题

**Q：是否需要联网？**
> A：需要联网，因为需要调用 DeepSeek API 进行实时分析。

**Q：使用是否免费？**
> A：是的！已内置阿里云 DeepSeek R1 API Key，使用阿里云百炼平台的免费额度。

**Q：语音输入不工作？**
> A：请检查以下几点：
> - 确保使用 **HTTPS** 环境（本地 `localhost` 除外）
> - 手机端**长按**麦克风按钮（约 0.4 秒），而非点击
> - 检查是否已允许**麦克风权限**
> - 推荐使用 **Chrome（Android）** 或 **Safari（iOS）**
> - 微信内置浏览器支持有限，建议切换系统浏览器

**Q：智能推荐为什么有时很快有时很慢？**
> A：预设商品（如手机）使用映射表，秒级响应；非预设商品调用 AI 分析，约 1-3 秒。推荐结果会缓存 1 小时。

**Q：手机端截图保存在哪里？**
> A：手机端点击截图后会弹出预览窗口，长按图片即可保存到相册。
